# 纯净互联网

所谓的算法入侵在每个媒体网站中，知乎从一个很好专业知识交流平台沦为一个垃圾新闻、垃圾问题的xx平台。如果你的血压比较低，是不需要使用这个插件的。

需要知道那么多信息吗？因此有了这个repo，针对大部分的社交媒体网站，只保留我关注的内容（如搜索、文章详情），去除推荐、热榜，对于无意义的评论模块也可能一并去除。
